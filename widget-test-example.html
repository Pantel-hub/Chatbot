Widget Integration Guide

Step 1: Get your API key from the Conferience dashboard

Step 2: Add the script tag to your website's HTML (before closing </body> tag):

<!-- Conferience Chatbot Widget --> <script src="https://your-conferience-domain.com/api/public/widget.js?key=YOUR_API_KEY"></script>

Step 3: Done! The chatbot will appear automatically on your website
